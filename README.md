# Customer Churn Prediction (Banking & FinTech)

## 📌 Project Overview
This project predicts customer churn using Machine Learning models.

## 🚀 Models Used
- Logistic Regression
- Random Forest

## 📊 Features
- Credit Score
- Age
- Balance
- Active Member
- Geography

## 📈 Results
- Logistic Regression: ~80%
- Random Forest: ~85–88%

## 🧠 Business Use Case
- Identify customers likely to leave
- Improve retention strategies
- Increase customer lifetime value

## 🛠 Tech Stack
- Python
- Scikit-learn
- Pandas
- Streamlit

streamlit run app.py
